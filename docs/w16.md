# 周叔

    *1997*

在这个家里，我当了整整七年的“周叔”。尽管在户口本上，我是敏敏的父亲，可在她心里，我只是“周叔”。

敏敏跟着秀兰嫁过来的时候，才六岁。小姑娘瘦瘦的，不爱说话，看人的时候总带着几分戒备。第一次见面，她躲在母亲身后，怯生生地喊了我一声“周叔”。

这一喊，就是七年，我从没逼她改口。有些东西急不得，树苗要慢慢长，人心也一样。

为了让这个重组家庭少一点别扭，多一点温度，我几乎把所有业余时间都搭了进去。敏敏喜欢画画，一个极其烧钱的爱好。一盒进口水彩颜料，抵得上我半个月的烟钱；一套画笔，能让我对着工资条发半天愁。

每周二和周五晚上，我都会骑着那辆除了铃铛不响、哪儿都响的永久自行车送她去市文化宫。冬天最难熬，北风从街口灌进来，吹得人耳朵生疼。我把军大衣反穿，把她裹在前面，像裹着个棉花团。她一路上缩着脖子，我就在后面蹬得满头热汗。

她进去画石膏像，我就在外面等。有时候门卫大爷心善，让我进传达室烤会儿炉子；大多数时候，我就蹲在路灯底下看借来的武侠小说，或者掏出烟盒纸，在背面一笔一笔算这个月还剩多少钱。两个半小时，她在里面学画画，我在外面学过日子。

后来她拿了少年组二等奖，兴冲冲地把奖状举到我面前，我高兴得像自己考上了大学。那天晚上特意割了半斤猪头肉，还买了瓶汽水。敏敏喝着汽水笑，我看着她笑，觉得这钱花得真值。

虽然穷，可我总觉得，日子是在往好的地方走。但前些天发生的事，让我的愿望彻底破灭了。

敏敏的初中学校里开始流行山地车。那种带变速器、前叉避震、车把像牛角一样翘起来的车，在孩子们眼里比轿车还神气。

那天晚饭时，她忽然说：“周叔，我想买辆捷安特，紫色的。”

我心里咯噔一下，一辆正版捷安特要八百多块。而我前两天刚交完暖气费，工资还没发下来，兜里连两百块都凑不齐。我放下筷子，尽量让语气听起来轻松一点：

“敏敏，八百块确实有点贵。爸去旧货市场看过，有辆翻新的坤车，两百多块，也是紫色的。买回来我给你重新喷漆，跟新的一样，行不行？”

敏敏抬起头，把筷子往桌上一拍：“我就知道！你除了会省钱还会干什么？人家王佳的爸爸也是继父，人家怎么就给买？”

继父！这两个字像锥子一样扎下来。七年里谁都没说过的话，终于还是说出口了。她摔门回了房间。

秀兰坐在饭桌边偷偷抹眼泪。我拍了拍妻子的手，什么也没说。

几天后，平静彻底被打破，秀兰的前夫回来了。那个男人消失了整整十年。听说这些年在南方倒腾电子元件发了财，开公司，当老板，挣了不少钱。他托人带话，想见见女儿。

秀兰气得发抖。“早干什么去了？现在想起来自己还有个女儿？”

我抽了一夜烟。天快亮的时候，对秀兰说：“让他见吧。血缘这种东西，拦不住。咱们越拦，敏敏以后越会怨咱们。”

见面的日子定在周五。我和秀兰特意请了假。她换上压箱底的呢子外套，我穿上那件平时舍不得穿的中山装。我们打算先去学校接敏敏，再一起去饭店。至少体面一点。哪怕只是一个穷爸爸最后的体面。

夕阳把校门口照得一片金黄。下课铃响起，学生们像潮水一样涌出来。忽然，秀兰抓住了我的胳膊，手冰凉。

顺着她的目光看去，我看见了敏敏站在人群中央，手里推着一辆崭新的紫色捷安特。车身上的英文标志在夕阳下亮得晃眼，周围围满了同学。她笑得那么开心，是我很多年都没见过的开心。

有个男生羡慕地问：“敏敏，这车谁给你买的？”

敏敏扬起下巴，声音又脆又响：“我爸买的！广州带回来的！我爸是大老板，这算什么！”

周围顿时一片惊叹。那个男生又问：“那经常接送你的那个呢？”

敏敏摆摆手，像是在说一件无关紧要的小事。“哎呀，别提了。那个小气鬼，谁管他。”

我站在人群外面，忽然觉得周围特别安静。耳边明明全是学生说话的声音，可我什么都听不见。

秀兰气得要冲过去。我一把拉住了她：“别去。”

我听见自己的声音沙哑得厉害。“孩子高兴。随她吧。”

说完，我转身推起那辆旧永久。链条发出咔哒咔哒的响声。和从前一样，只是车还是那辆车，人却好像突然老了很多。

一路上，我想起很多事。想起文化宫门口的路灯，想起冬天军大衣里的热气，想起她拿着奖状扑进我怀里的样子。那些我以为会一直留下来的东西，原来并没有我想象中那么牢靠。后来我终于明白了一件事，养育和血缘，从来不是一场公平的比赛。血缘是与生俱来的礼物，而养育，更像是一份漫长的工作。它没有证书，没有奖章，也未必能得到回报。你只是日复一日地付出，然后等待。有时候能等来一句“爸爸”，有时候什么也等不到。
