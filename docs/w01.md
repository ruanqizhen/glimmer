# 可以缺席的文学课

那几年，“下岗”成了最常被提起的词。工厂的公告栏成了危险地带，每天总有人装作路过，然后不自觉地放慢脚步，目光在红纸边缘游移，生怕看见自己的名字。

我在供销科守着无人问津的冷板凳，心已经悬到了半空。直到“学历即护身符”的流言传遍了办公室，我才如梦初醒，一脚踏进了夜大的校门。

夜大的好处在于，它不问你为何而来，只确认你是否缴费。我选了汉语言文学专业，并非出于热爱，而是因为文学对人最宽容：它不需要令人费解的数理化，只要识字，就能被接纳。文学当然解决不了生计，但它至少能让生计显得不那么狼狈。

第一门课却是《现代西方文学》，由孙教授主讲。早年他专门研究无产阶级革命史，一生致力于批判资本主义；冷战结束后，年近花甲的他，却不得不站在夜大的讲台上，向一群为了饭碗而来的成年人讲卡夫卡和萨特。

“格里高尔变成了一只甲虫，”他拍着讲义说，“这是一种深刻的异化。”

讲台旁，他的旅行包敞着口，露出几本新书的书脊——《存在主义与现代灵魂》，作者正是他自己。下课前，他总会顺口提醒一句：书中某某章关于某某问题的论述，是期末考试的重点。

话说得很克制，也说得清楚。于是大家心领神会，排队交钱。

既然知识可以用现金结算，出勤率自然也就失去了意义。夜大的逻辑非常简单：钱留下，人爱去哪儿去哪儿。我很快掌握了这套规则，开始时不时地逃课。

期末将近，班长开始私下收钱，说是给老师送点心意。名义上是敬老，实际上是买平安。大家都明白，也都配合。

考试那天，教室灯火通明，气氛却带着一种刻意维持的庄严。监考老师象征性地在过道里走了两圈，随后在讲台边坐定，翻开一本杂志。剩下的时间，便交给我们自行处理。

考生们各显神通。有人把课本缩印成纸条，有人把答案写满手心。我们像一群临时凑齐的地下人员，用最粗陋的手段，从事一场自以为高雅的行动。

我左手边坐着一个陌生的年轻人。他穿着剪裁得体的风衣，握着一支派克钢笔，神情悠闲。他显然没做好准备，卷面一片空白。我出于一种底层互助的本能反应，把答卷往他那边挪了挪。

他看了我一眼，小声说：“不用了，多谢。”

出于礼貌，考后他同我聊了几句。他说他父亲前几天刚和校办主任谈过“文学艺术对企业文化建设的指导意义”这个话题。这些课程的设置，都是他父亲参与策划的。他坐在这儿，只是为了让这张试卷显得有人作答。

那一刻我才意识到，我们从一开始就不是同学。我捏着手里那支廉价的圆珠笔，忽然感到一种异常清晰的荒谬：我以为靠一点小聪明，可以戏弄教育的形式主义；却直到此时才明白，真正有资格戏弄规则的人，根本不必进教室。
