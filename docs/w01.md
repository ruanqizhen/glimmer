# 解不出的题目

小芳身材高挑，走路时马尾辫轻轻摇曳，偶尔扫过白衬衫的领口。那衬衫总散发着淡淡的肥皂清香，像是刚在阳光下晾干。她喜欢穿一条洗得发白的牛仔裤，裤脚总是短一截，露出纤细的脚踝。她回眸的瞬间像一缕微风，吹进了死气沉沉的教室，成了无数男生在课本边角偷偷写下又匆匆划掉的名字。

少男少女聚在一起，话题总绕不开“谁喜欢谁”。俺们东北那儿的人性子直，有什么心事，不会藏着掖着。但我很少掺和这些闲聊，因为我脑子里只装着一个念头——去上海。

高一结束后，父亲遇到了一个难得的机会，因公出差去上海。家里并不富裕，他反复盘算了好几天，最终还是咬牙给我买了张火车票，带我去见见世面。那趟行程彻底颠覆了我对生活的认知。我望着黄浦江畔一幢幢灯火通明的楼宇，怎么也数不清；南京路上，西装革履的人们匆匆穿梭，怎么也追赶不上时间。霓虹闪烁、玻璃幕墙、拥挤的人潮、川流的车河，都让我这个来自穷乡僻壤的少年目眩神迷。

从上海回来后，镇上的一切都显得狭小了。街道依旧，可我心里却像被什么堵住了似的。我清楚地知道，自己必须离开这里，去上海，去更远的地方。而通往远方的独木桥，只有一条：高考。必须考上上海的顶尖大学，没有退路，不能妥协。

成绩好的学生在学校总是更受欢迎。我虽然不是班里排名第一，却是人缘最好的。因为无论谁来问问题，我都会耐心地讲解，直到他们弄懂为止。哪怕只是想抄作业，我也不会拆穿，直接把本子递过去。

正是在一次次帮助同学的过程中，我开始感受到人与人之间的差距。对我而言，解题如吃饭喝水般自然，对另一些人来说却是翻不过去的坎儿。小芳，就卡在那道坎儿上。

高三分座位时，她被调到了我的正后方。以前没怎么说过话的我们，从此有了交集。她的存在忽然变得真实而亲近：发丝拂过时的淡淡香气，写字时脚不经意碰到椅腿的轻响，还有她常常拿着卷子，脸颊微红地轻敲我的后背：“这道题我又不会了，能帮我看看吗？”

我会转过身，尝试用各种方式拆解那些复杂的数学公式，一步一步，替她铺出能攀登的阶梯。

她是班里最勤奋的学生之一，只是数学天赋欠缺。我记得有次她做错了一道解析几何题，我给她讲了三遍，她依旧不明白。她急得眼眶都红了，咬着嘴唇说：“我是不是特别笨？”我说：“不是，是我没彻底吃透这道题，所以讲不清楚。我们再换个思路试试。”她抹了抹眼角，深吸一口气：“嗯，你再讲一遍，我一定能懂。”那天我不知道讲了多少遍，一直到晚自习结束，她终于把答案算对时，整个人几乎跳了起来，兴奋得像个被表扬的小学生。

后来我发现，她有时会走神。咬着笔帽，眼睛望着窗外发呆。我提醒一句，她立刻回过神来，脸一红，低头继续算题。

有时候，她会悄悄在我桌角放一颗水果糖。我拿起来放进嘴里，她就趴在桌上假装看书，耳朵根却已经红透了。

有些东西，就在数字的计算和公式的推演里，悄然变了味儿。

最先失控的是我的同桌，也是我曾经最好的哥们儿。那天教室里只剩我们俩，他突然停下笔，极其委屈地控诉：“我忍不了了。你俩都当我是透明的，每天在这里卿卿我我。你明知道我以前追过她的。你们从来没有考虑过我的感受！”

第二天，他找班主任调了座位，去了教室最后面的角落。眼不见心不烦。

一石激起千层浪，流言很快传开。班主任看她的眼神也变了：小芳曾因容貌出众而备受偏爱，现在却成了不务正业，还会拖累优秀学生的坏榜样。没多久，班主任随便找个借口，把她也调远了。

物理距离拉开了，心理距离却没有。只要是课间或自由活动的时候，她依然会准时出现在我桌前。我们聊的始终是数理化，可在外人看来，我们俩总黏在一起，早被默认成了一对。

有一天，另一个哥们儿在厕所里把我堵住了，他眼睛里闪着受了刺激的狠劲儿：“你以为我真拿你当朋友了？我就是想借机靠近她……你以后要是敢欺负小芳，我饶不了你……”

更尴尬的是，一天中午，操场上阳光明媚，我被几个女生拉去打羽毛球，正打得起劲，小芳突然气呼呼地冲了过来，当着所有人的面冲我大喊：“那两道题你还没教会我呢，怎么就跑出来玩了！”

满场一片寂静。几个女生面面相觑，然后找了个借口，一哄而散。

下午自习课的时候，我的新同桌和后排的女生嘀咕了半天，然后一起转过头来，一本正经地问我：“我们以后还能跟你说话不？不会被小芳‘追杀’吧？”

我只能回以一个无奈的苦笑：“我俩没什么，只是在帮她补习而已。”两个小丫头立刻恢复了往日的嘻嘻哈哈，说：“那太好了，这道题到底咋回事，赶紧给我们讲讲。”

她的心思越明了，我的内心就越胆怯。毕竟，我从身边的每个家庭里感受到的，都不是爱情的美好，而是贫贱夫妻百事哀，是在昏黄的灯下为几十块钱吵得面红耳赤。我害怕自己也陷入那般境地。以我的家庭条件，若走不出这个小镇，眼前的甜蜜终究会化作将来的一地鸡毛。

我挣扎了许久，最后还是上海赢了。我下定了决心，不能让任何羁绊挡住我逃离的路。

从那天起，我开始刻意回避。她再来问问题，我依然耐心，却会故意隔着桌子，目光只盯着卷子，绝不与她对视。又一次悄悄出现在桌角的水果糖，我也没有放进嘴里，而是当着她的面，塞进了书包。

我不知道她是什么时候察觉到了我的疏远。她还是会来问问题，但频率明显少了。有时候明明拿着卷子走到我桌前，却又犹豫着退回去。或者只是在我路过她的桌子时，才小声说一句：“如果不耽误你的话……”

在彷徨与煎熬中，高三一晃就过去了。我拿到了通往魔都的录取通知书，而小芳则去了本地一所大专。以她的基础，这已是极限，我也尽力了。

在那个交通基本靠走、通讯基本靠吼的小镇里，离别显得格外决绝。我想过去她家道个别，可走到半路又停下了。既然什么都承诺不了，还去说什么呢？说“我要走了”？还是说“祝你幸福”？

还是算了吧。她没有来找我，我也没有再打扰她。

大学毕业后，我出国深造，越走越远，与过去的圈子逐渐失联。直到多年后，互联网普及了，我才又零星听到一些她的消息。她留在镇上，结婚生子，日子安稳。这是多么圆满的结局：我终于见识了世界的辽阔，她也在小镇找到了自己的归宿，我们都活成了各自向往的样子。

可是，我还是拿起了桌角的玻璃瓶，里面藏着一颗，我舍不得吃掉的水果糖。
