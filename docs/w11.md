# 半个红薯

林子常向人讲述我们那段“红薯情谊”。他说得绘声绘色，动情处还会拍桌子：“要不是大壮，当年我早就没了！”

那是 1960 年的冬天。风从土坯房的裂缝里钻进来，像刀子一样割脸。村里的树皮早被剥光，连野草根都翻了个遍。人饿到后来，肚子不叫了，腿却肿得发亮，手指头一按一个坑，半天弹不回来。

林子病得最重。躺在炕上发高烧，说胡话，嘴唇裂得像干旱的田地。他娘早没了，他爹也没撑住走了，只剩我陪着他。那天夜里，他迷迷糊糊抓着我衣角，小声叫水。可缸里只有半瓢浑浊的雪水。

我跑去后山。那片地早被翻过无数遍，像一张被撕碎的席子。我也不知道为什么还要去翻，大概是人到了绝路，总还想赌一把。天快黑时，我在一块冻得发硬的土坷垃下面，摸到一个硬疙瘩。

是个红薯。

拳头大，干瘪，皱得像老人的脚后跟。可那一刻，它就是命。

我揣在怀里，用体温焐着。没敢生火，怕烟味惹来人。回到屋里，我拿石头把它砸开。我把大的一半塞进林子嘴里：“嚼。阎王爷打瞌睡漏下的，吃了咱俩都能活。”

他睁开凹陷的眼睛，愣了一下，然后狼吞虎咽，连皮带泥一起吞了。眼泪顺着太阳穴往下淌，嘴里含糊不清地说：“大壮……这命是你给的。以后我有口干的，绝不给你喝稀的。”

那时候，我真觉得，我把半条命分给了他。

后来日子慢慢好起来，我们一块下地干活，一块娶妻生子。谁家有事，另一个准到。那种情分，不用说，像埋在土里的老根，平时看不见，却总在。

八十年代末，世道变了。林子脑子活，胆子大，成了村里第一个“下海”的人。他南下广东倒腾服装，后来又做皮具。信一封封寄回来，说生意不错。再后来，信少了。但是听说他发了，坐的是大奔，用的是“大砖头”手机。有人说他在广州有公司，有人说他在深圳有厂房。

那年夏天，太久没见他了。我想了很久，还是坐上了去广州的绿皮火车。车厢里闷热，泡面味混着汗味。我抱着一个帆布包，里面装着几斤自家晒的花生——总不好空手。

我在写字楼里见到了他。大理石地面亮得能照出人影，空调冷得我直打喷嚏。秘书端来一杯褐色的酒，说是“威士忌”。我没喝出什么味道，只觉得辣。

林子穿着西装，领带打得一丝不苟，头发不多了，却梳得整整齐齐。他拍着我肩膀说：“老兄弟啊，多少年了。”说话时，他的手机响了两次，他按掉了。

我们聊过去，聊村里人。他笑得很大声，像在回忆一个遥远的故事。那个红薯，他也提了，说得轻描淡写：“那时候真苦啊。”

临走时，他递给我一个牛皮纸口袋。

“大壮，这里两千。回去把房子修修。”

两千，对一个还在种地的农民来说，是笔大钱。我握着那口袋，心里却说不清是什么滋味。

我忽然想起那半个红薯。那不是钱，是命，是两个少年在风里抱团取暖。那时我们什么都没有，只有彼此。现在，他什么都有了，而我却成了他两千块钱的一个交代。

我把口袋放回桌上。

“林子，我不是来要钱的。”

他愣了一下，脸色沉下来：“大壮，你什么意思？嫌少？我不是外人，这已经很够意思了。”

他声音有点高，秘书在门外探头。

我忽然觉得冷。不是空调的冷，是那种说不清的凉。

“那半个红薯，”我慢慢说，“我没算过价。”

空气一下子静了。他张了张嘴，终究没说话。

我转身走出那栋大楼。玻璃门在身后合上，像把两个世界隔开了。
