# 忠狗

我在市商业局人事科科长这个位置上坐了整整五年。那是我人生最风光的五年，也是最糊涂的五年。

当时，我手里那支红蓝铅笔，就是整个系统的“指挥棒”。想调进百货大楼当售货员？想批条子去物资公司拉钢材？机关食堂想换个采购员？一句话，我说行就行。我的办公室门槛几乎被踏破，每天晚上的饭局排到后半夜，茅台酒瓶子在我家阳台上堆得像一堵墙。

我慢慢习惯了被人围着转的感觉，习惯了别人喊“赵科长”时那股子小心翼翼、带着讨好的劲儿。人一旦尝过权力的甜头，就很难不飘了。

苏晓梅是局里从县文工团借调来的广播员。中午十二点半，机关大喇叭准时响起，她那口清亮标准的普通话，像一股凉风，把满楼的湘西、豫东、皖北口音全压下去。她长不算多漂亮，但年轻就是好啊。那友谊雪花膏淡淡的甜香，一闻就让人心里发软。县文工团快散伙了，她想留在城里。要留下，编制、户口、农转非，三样缺一不可。

她第一次敲我办公室的门，是拿着一份播音稿。“赵科长，您帮我看看这段，是不是太生硬了？领导听了一定不高兴。”
门虚掩着，午休时间，走廊安静得能听见苍蝇撞玻璃。她低着头，红着脸。我知道她为什么来，她也知道我知道，谁都不用把话说破。很多事情，本就只隔着半步——你往前一点，我就顺势而为。我享受那种被仰视、被依赖的感觉。她看我时眼神柔软，带着一点讨好。

她的编制问题，我一直压着。其实并不难办，但办得太快，人未必还会留在我身边。那时的我，把这种拿捏当成手段和智慧。现在想来，真是愚蠢。

王志远是分配来的大学生。大学生在我们那是稀缺资源，但进了机关，一样要从写材料做起。他戴副黑框厚眼镜，话少，写文件爱用长句子，动不动“在……的前提下”“进一步强化……的重要性”。我看不惯，机关最怕的就是“领导看不懂”。我骂过他无数次，当着全科人把材料摔桌上：“谁看得懂你这狗屁不通的东西？重写！”

他每次都弯腰一张张捡起来，声音低得像蚊子：“对不起科长，我马上改。”

他就这点好，像条忠心的狗。于是我渐渐把他当心腹，私下里让他帮我办一些我不好直接露面的事情。

有一天下午，我正和小苏在办公室里，准备“深入讨论一下工作”。突然门被推开——王志远抱着文件夹，愣在门口。

我和小苏的保密工作做的非常好，王志远也没有发觉过。这次被他撞见，我心里一阵恼火，顺口就骂：“没正经事干吗？上班时间到处乱窜。这个月总结到现在还没交，再不交你这个月奖金别想了！”

苏晓梅在沙发上轻笑出声。王志远抬头看了我们一眼，脸色煞白，喉结滚了滚，只说了一句：“对不起，我这就去写。”转身就走。

我没放在心上。在我眼里，他不过是一条需要我喂饭的狗。

那年局里有一个内部“农转非”的名额。苏晓梅盯得很紧。那天晚上在宾馆里，她问我：“老赵，这次一定是我，对吧？”

我抽着烟，半天没说话。“这次……局长外甥已经内定好了。下次我保证。”

她没有哭，只是安静地穿好衣服，背对着我说：“你每次都说下次。”

她走的时候，高跟鞋在走廊里响得很慢，却始终没有回头。

后来我发现王志远三天两头往广播站跑。我问他：“小王，最近很闲吗？老往那边跑什么？”

他笑着说是帮忙修广播线路。那笑容依旧憨厚老实。

我心里暗自揣测，他怕不是也对苏晓梅动了心思。这对我到底是好事呢，还是坏事呢？我一时间理不清头绪。等我意识到不对的时候，一切都晚了。

两个月后，市纪委的人直接进了我的办公室。带头的人语气平静：“赵建国同志，接到实名举报，请配合调查。”

举报材料很厚。最上面，是苏晓梅的亲笔信。时间、地点、用了什么姿势，一条条写得清清楚楚。字迹是她的，但长句子、逻辑、标点，全是王志远那套“学院派”笔法。附件里还有十几张照片——角度刁钻，全是远处偷拍，边缘还带着日期。

我一下子明白过来：那些“修广播线路”的下午，那些低头捡材料时的沉默，全是他布的局。

半年的审查，我像被剥掉了一层皮。虽然没有查出确凿的经济问题，但“生活作风严重腐化”已是铁板钉钉。处理结果：留党察看；撤销一切行政职务，调到局属仓库看管物资，每天和垃圾、老鼠、霉味打交道。

在我被审查期间，小苏被局里开除，遣返回县文工团。王志远则被提拔为副科长。

后来有一次，我在单位大院里清理东西。正巧王志远路过，看到我，就一脸笑容走了上来，拉着我的手说：“老赵啊，好久没见了，前几天还想去看看你呢，但是工作太忙了，最近怎么样啊？”那笑容依旧憨厚老实，看不出半点复仇的快意，也没有幸灾乐祸，更没有一丝怜悯。
