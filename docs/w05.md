# 批判相对论

五十年代初，我在一所中学教物理。那时候，教学大纲、参考资料，乃至思维方式，几乎全部来自苏联。谁也不觉得这有什么问题。

1953 年初的一个清晨，我在简陋的教研室里翻看《人民日报》时，发现报纸上转载了苏联方面的文章，措辞严厉，明确指出爱因斯坦的理论“具有唯心主义倾向”，并由此上升为意识形态问题。那一刻，我心里“咯噔”了一下。既然苏联已经定了调，那么对爱因斯坦的态度，就不再是学术问题，而是立场问题了。

那天晚上，我把备课本里关于爱因斯坦的赞语一一划掉。此后上课时，每当提到爱因斯坦，都会特意补上一句：“爱因斯坦在世界观上属于资产阶级。”

六十年代中期，我因为在私下聊天中流露出对爱因斯坦学术贡献的认可，被人提醒“立场不够鲜明”。有同事转告我，上面正在注意这些问题。我这才意识到，之前的谨慎还远远不够。

一次会议上，学校传达了报刊文章的精神，强调爱因斯坦在政治上的“错误立场”，包括他曾支持美国研制原子弹一事。连一些知名科学家，比如钱学森，也都撰文表态，批判其思想倾向。我心里很清楚：既然连这些人都得表态，我这样一个普通的中学教师，更没有沉默的余地。

那段时间，我把家里保存的几本外国物理学资料陆续处理掉，大部分都送进了灶膛。只有一本《相对论》，我始终没舍得烧。它被我用旧报纸包好，悄悄藏了起来。

对相对论的公开批判，在“文革”期间达到了高潮。1967年前后，各地开始用毛主席语录批判物理学问题。随后，中科院发表文章，系统批判“光速不变原理”，声称这一观点影射了资本主义制度不可超越。

那天夜里，我躺在床上自言自语：“老林，从明天起，光速就可以变了。如果组织上要求一加一等于三，那它就必须等于三。”

不久，这样的心理准备果然派上了用场。在一次政治学习会上，一位年轻教师当众问我：“林老师，相对论在珍宝岛问题上适用吗？”

我愣了一下，冷汗顺着后背往下流。我扶了扶眼镜，按照早已想好的说法照本宣科地回答道：“任何科学问题，都要放到阶级立场上去看……”


1979 年春天，我在收音机里听到一条新闻：科学界在北京召开会议纪念爱因斯坦诞辰一百周年。我听后终于松了一口气。

那天下午，我从衣柜的夹层中取出了那本藏了十多年的《相对论》。书页已经发黄，封面生了霉斑。我翻开它，发现爱因斯坦还是那个爱因斯坦，光速还是那个光速。
