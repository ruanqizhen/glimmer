# 批判相对论

五十年代初，我在一所中学教物理。但我对相对论的兴趣，其实更早就开始了。

相对论在二十世纪初便已传入中国。1919 年日食观测验证广义相对论后，爱因斯坦声名大噪，中国知识界很快通过英文、日文相关资料接触到这一理论。五四时期，它几乎成为“现代科学”的象征。三四十年代，一批中国学者如周培源等人在海外研习理论物理，把相对论带回国内。那时，在我们学生眼中，爱因斯坦不仅是科学家，更是理性与创造力的象征。

我在求学期间曾借到过一本介绍相对论的小册子，读得入了迷。光速不变原理、时空观念的革命性变化，让我第一次意识到，科学并非对常识的修补，而是对世界图景的重塑。

新中国成立初期，教学体系大量借鉴苏联模式。物理教材、参考书乃至课程结构，都深受影响。这本身并不奇怪——苏联在基础科学领域确有深厚积累。

五十年代初，苏联哲学界内部曾出现对相对论的批评声音，认为其在哲学解释上存在“唯心主义倾向”。这些讨论陆续被介绍到国内。有的报刊转载相关观点，强调科学理论必须以辩证唯物主义为指导。

1953 年初的一天，我在教研室读到了相关文章，心里一沉。我隐约意识到，从此对爱因斯坦的评价，很可能将不再只是学术问题，而是会牵涉到立场问题。作为中学教师，我只能在课堂上小心措辞。提到爱因斯坦时，我都会强调一下“理论的哲学解释需要用唯物主义观点加以分析”。

事实上，即便在苏联，相对论也并未被正式否定；批评更多停留在哲学层面。但在当时的氛围中，很多人搞不清楚这里边的差别。

进入六十年代，政治气候趋于紧张。科学讨论越来越容易被赋予意识形态意义。爱因斯坦的政治立场也被反复提及，例如他在 1939 年签署致罗斯福总统的信中，建议关注原子能军事潜力；又如他晚年支持世界政府与和平运动。这些复杂背景常被简化为批判他的标签。

“文化大革命”爆发后，学术问题更直接卷入政治斗争。部分刊物出现批判相对论的文章，把“相对性”与“相对主义”混为一谈。有些基层单位也组织过学习讨论，把物理理论纳入政治语境。

但与后来一些流传甚广的说法不同，并不存在全国范围内由官方正式宣布“推翻光速不变原理”的决定。更多是局部单位、个别人借政治风潮发表极端观点。科学界整体研究虽受冲击，却并未在理论上另立体系。

对我们普通教师来说，压力是真实的。我记得一次学习会上，有年轻同事半开玩笑地问我：“林老师，相对论是不是资产阶级的东西？”我愣了一下，只能回答：“科学理论要放在实践中检验，也要结合历史条件来看。”这种模糊的表达，这是当时许多人不得不用的自保方式。

那段时间，家里珍藏的几本外文书籍都被我处理掉了。我不是怀疑它们的价值，而是害怕会给家人带来麻烦。但唯独那本《相对论》没有舍得丢，我用旧报纸包好，把它藏了起来。

七十年代初，局势开始微妙变化。1971 年后，国家逐步恢复对科研工作的重视。1972 年，全国科学规划会议召开，提出要把科学技术搞上去。基础研究虽仍受限制，但空间正在缓慢恢复。

1978 年，全国科学大会召开，明确提出“科学技术是生产力”。次年，中国科学界举行活动纪念爱因斯坦诞辰一百周年。会上公开肯定相对论在现代物理中的基础地位。这不仅是对一位科学家的评价转变，也是对一个时代学术环境的修正。

那天，我从衣柜夹层里取出那本藏了多年的书。纸页发黄，封面起了霉点。我翻开它，发现爱因斯坦还是那个爱因斯坦，光速也依然是那个光速。
