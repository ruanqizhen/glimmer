# 蓝印

我中学毕业那会儿，正赶上深入改革开放的年月。农村承包制已经稳定下来，集贸市场重新活跃，个体经营越来越多。我们这种庄户人家，也不再全靠种地过日子。

我顺理成章接了父亲的屠户行当。收猪、杀猪，在乡镇算是正经买卖。那几年，镇上鼓励“勤劳致富”，还给我评过“致富标兵”。照片登过一次乡镇小报，算是风光了一回。做生意讲的是口碑，我收猪从不压价，卖肉也不缺斤短两，更不做注水掺假的事。乡亲们信得过我。

谁成想，我最后栽在一个印章上。

那是国庆节前一周。节前肉好卖，我天没亮就宰了几头肥猪。按当时的规矩，生猪上市前要经过检疫。镇防疫站派人到场查验，合格的猪胴体上要盖蓝色检疫章，才能进集市。

那天来的是防疫站的李干事。他扫了几眼，又象征性翻了翻内脏，就拿出印章在猪胴体上按了几下。蓝色印泥有些发干，印迹模糊，只能看见个边框。

我弯着腰陪笑：“李干事，麻烦您再盖清楚点吧，印子太淡，到了集上不好说话。”

他叼着烟，不耐烦地挥手：“合格就是合格，我人都来了，还能有假？别磨蹭，我赶时间。”

我心想，人家是公家的，又收了检疫费，应该不会出岔子，也就没再坚持。把肉装上三轮车，赶往集市。

那几年农村集市很热闹，但检查也比从前严。私宰乱卖是要罚的。摊子刚摆开，生意就火起来。熟客一边称肉一边闲聊，一切都和往常一样。

偏偏那天出了事。

老张家小子回去吃完肉，上吐下泻。家里急了，报了派出所。没多久，执法人员来到集市检查。他们一眼就盯上那头印章模糊的猪肉。蓝戳看不清，便当场认定检疫手续存疑，怀疑是私宰。

我急忙去找李干事作证。他却翻了脸：“我们站里盖章都很清楚，这肉哪有合格章？我没给这头猪检过！”

他说得干脆利落。我愣在那里，一句话也接不上。那时候，基层执法还谈不上规范取证，谁的话算数，很清楚。

我的农用三轮车被当场扣下，肉也封存。最后认定为“未经有效检疫销售肉品”，罚款五千元。

一头猪才卖七八百块，五千块，跟天塌下来差不多。后来听说，老张家孩子只是急性肠胃炎，未必和我的肉有关。
