# 一个印章

我中学毕业那会儿，正赶上改革开放的好光景，就顺理成章接了老爹的屠户行当。咱庄户人做买卖，讲的就是个踏实本分。我收猪从不短斤少两，杀猪绝不注水掺假。前几年镇上还把我评成“致富标兵”，照片登在报纸上，在镇里也算露过一回脸。

谁成想，我最后栽在一个小小的印章上。

那是国庆节前一周。为了赶集，我天没亮就宰了三头大肥猪。按规矩，早早把防疫站的李干事请来检疫。他那天穿得挺整齐，像是要去县里吃席，嘴里叼着烟，走到猪跟前看了两眼，就在它们屁股上草草按了个章。那蓝印泥估摸着快干了，盖出来的印子虚虚的，只能勉强看出个边。

我弯着腰，陪着笑递过去一根“红塔山”，压低声音说：“李干事，您再帮着盖清楚点吧，这印子太淡，怕到集上不好说。”

他斜我一眼，啪地一声点着火，吐了口烟，不耐烦地挥挥手：“你操啥心？我亲眼看过，合格就是合格！费你也交了，我人就在这儿，还能出啥事？赶紧卖你的去，别耽误我。”

我心想，人家是公家的，公家的话还能有假？也就没再多说，拍拍裤子上的灰，把肉装上车，高高兴兴往集市赶。

摊子刚摆开，买卖就火起来了。乡亲们都熟门熟路，排着队称肉，说说笑笑。

偏偏那天出事。

老张家那小子回去吃完肉，闹起了肚子，上吐下泻，家里人一口咬定是肉有问题，一气之下报了警。没一会儿，大队民兵和派出所的人一块儿来了，直奔我摊子。

一查，就盯上那头印章模糊的猪。那蓝戳看不清样子，他们当场认定是私宰，说没经过正规检疫。

我急着去找李干事作证，谁知道他当场翻脸：“我没给这头猪盖过章！我们防疫站的规矩是见肉盖章，这猪身上哪有清楚的蓝戳？分明是你想省那几块钱，自己私宰的！”

他说得斩钉截铁，我一句话都插不上。

我百口莫辩。农用三轮车被当成“作案工具”当场扣走，还罚了五千块钱。一头猪才卖七八百，五千块对我来说，跟天塌下来差不多。
