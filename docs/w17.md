# 去美国的船票

   *1995* 

满大街的理发店、大排档，还有修摩托车的铁皮棚里，都挂着方方正正的音箱。音箱音质很差，放出来的歌却特别响。有时是《牵手》，有时是《祝你平安》，有时是我不知道名的情歌。歌声从街头飘到街尾，和工地上的打桩声、汽车喇叭声混在一起。

深圳这几年变得太快了。

我刚来那会儿，罗湖很多地方还是荒地。如今深南大道已经修得宽宽敞敞，两边却依然到处是工地。塔吊一天到晚转个不停，运泥车来来往往，空气里总飘着水泥和红土的味道。

每个人都很忙。卖服装的忙，开工厂的忙，跑运输的忙。好像谁停下来，谁就会被这座城市甩在后面。

我已年过三十，这些年一直在东门做服装批发生意。每天凌晨三点多起床。洗把脸，啃两个包子，拎着蛇皮袋赶去火车站。

最早一班去广州的火车永远挤得满满当当。车厢里到处都是做生意的人。有人卖服装，有人卖鞋，有人卖电子表。过道里堆着编织袋，座位底下塞着纸箱子。空气里混着方便面、茶叶蛋和汗味。

有时候站两个小时都找不到地方坐。可没人抱怨。因为大家心里都清楚，下了车就是钱。

靠着这样一年到头地跑，我总算攒下了一点家底。我床底下放着一个旅行箱，里面装着我这些年的积蓄。一捆一捆的百元钞票，用橡皮筋扎得整整齐齐。

晚上睡不着的时候，我偶尔会把箱子拖出来，坐在床边数钱。数完一遍，再数一遍。不是怕少，就是心里踏实。我知道自己在深圳站住脚了。

空闲的时候，我喜欢拉着好友阿芳去逛街。但我身边一直带着 BP 机。BP 机一响，我就要赶紧找电话回过去。有时候是客户催货。有时候是广州档口来了新款。做生意的人最怕错过消息。

年初，我认识了理查德·梁。那天阳光酒店举办一场经贸联谊会。听说来了不少港商和华侨老板。我托朋友要了一张入场券。我想去那儿看看，有没有机会能把服装卖到香港去。

那天下着雨，酒店门口正在施工，地上一片泥泞。我刚下出租车，一脚踩进泥坑里。泥水溅得到处都是，高跟鞋也脏了。

我正蹲在路边拿纸巾擦鞋，一辆黑色丰田皇冠缓缓停在旁边。车窗降下来,里面坐着一个男人,三十多岁，戴金边眼镜,皮肤很白。

他递给我一把长柄黑伞，用带着一点台湾腔的普通话说：“雨这么大，别淋感冒了。”

我拿着伞，一下子愣住了。等回过神，他的车已经开走了。

但我怎么也没想到，能在会场里再遇到他。

他主动过来打招呼：“刚才没把鞋弄坏吧？”这便是我们的第一次正式交谈。

他说他叫理查德·梁，长期住在美国，目前在一家投资公司工作。这次回国，是来寻找合作项目的。

联谊会上到处都有人交换名片，老板们说话声音特别大，喝酒也凶。可理查德不一样，他说话总是不紧不慢。别人敬酒，他点到为止。别人吹嘘自己赚了多少钱，他只是微笑着听。那种从容，在生意人中并不多见。

后来我们又见了几次面，有时在酒店喝咖啡，有时在罗湖的茶楼吃早茶。他见识很广，从美国聊到香港，从股票聊到房地产，从服装出口聊到国际贸易。很多我不懂的东西，他都懂。

有一次在罗湖一家西餐厅，他说：“曼，你知道吗？像你这样的女人，如果一直守着东门那个档口，太可惜了。”

我问：“那你觉得我该做什么？”

他说：“应该做自己的品牌。把衣服卖到美国去。”

那一刻，我承认自己动心了。这些年我拼命挣钱，可心里始终有个遗憾。我再有钱也只是个土掉渣的个体户。而他描述的未来，却像另一种人生。

后来我们的关系越来越近他经，常带我去阳光酒店顶层喝咖啡。晚上从窗户往外看，深圳灯火通明。远处还能看见香港方向的灯光。

他说：“跟我去旧金山吧，那里天气很好，不像深圳这么闷热。你会喜欢的！”

说这些话的时候，他总是特别认真。认真到让我无法怀疑。

我从来没有出过国，甚至连近在咫尺的香港都没去过。在我眼里，美国远得像另一个世界。而理查德，就是连接那个世界的人。

阿芳提醒过我，让我提防这个人。有一次她把我拉到一边，悄悄问：“你查过他吗？”

我有些不高兴：“查什么？”

“他说的那家公司，我怎么没听过？”

我笑了：“美国那么多公司，你能听过几个？”

阿芳皱着眉：“反正我觉得他这个人不太是在。”

我当时根本没把这些话放在心上。后来想想，有些事情，旁观的人往往比当局的人看得清楚。

九月中旬，理查德忽然变得焦虑起来，连续几天都没怎么睡好。有时半夜还在房间打国际长途。

我问他出了什么事。

他说：“有个项目马上能批下来。只差最后一步。”

几天后，他拿来一份厚厚的英文文件，告诉我美国那边已经同意合作。只要缴纳一笔保证金，项目就能正式启动。

他说：“这是手续问题。资金只是暂时冻结，等办完以后全部返还。”

我问他还差多少钱，他说八十万。

听到数字的时候，我心里咯噔一下。我这些年的全部积蓄也才五十多万。

可他却安慰我说，钱的事情不要我担心，他能搞定。正因为这样，我反而更相信他。

我把床底下的全部积蓄都搬了出来，又找亲戚朋友借了一部分，帮他凑齐了八十万。

钱交出去那天，理查德抱住我。他说：“相信我。这是通往新生活的第一步。”

我信了，完全信了。

十月初，理查德说要去广州见一个重要人物。如果顺利的话，回来就可以开始办理结婚和出国的手续了。

临走时他站在门口，穿着那件米色风衣，回头冲我笑了笑，说：“等我回来。我们去拍结婚照。”

那是我最幸福的时刻，也是最后一次见到他。

第一天，没有任何电话。第二天，BP 机没有回复。第三天，我开始慌了。

我跑去阳光酒店，他平时长住的酒店。前台告诉我，梁先生早已退房了。

我纠结了很多天，最后，还是鼓起勇气去了派出所。

负责接待的民警听完经过以后，丝毫没有惊讶。他翻了翻资料，又打了几个电话，最后抬头看着我，说：“这是一种很常见的骗局，最近已经有过多起类似的案件了。”

我只觉得耳边嗡的一声，后面的话再也听不清了。走出派出所时，太阳很大。深南大道上车来车往，喇叭声不断。工地上依然尘土飞扬。

深圳还是那个深圳，好像什么都没变，又好像什么都变了。

我一直在路边坐到天黑，看着街上的人来来往往，忽然觉得特别累。这些年吃过那么多苦，扛过那么多货，熬过那么多夜，我从来没哭过。但这一刻，我觉得自己再也坚持不住了。

我去了阿芳的面馆，她看见我的样子，什么都没问，只是做了一碗牛杂面放到我面前。牛杂热气腾腾，辣椒油漂在汤面上。她说：“先吃东西吧。”

我低着头吃了一口，眼泪掉进碗里。忽然觉得特别辣，也特别苦。
