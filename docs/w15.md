# 坪石夜车

1996 年的深秋。我们几个同村老乡在广州花都结清了泥水工的工钱，把铺盖卷和工具塞进蛇皮袋，挤上了一辆开往湖南衡阳的长途客车。

那是一辆“黄河”牌大巴，方头方脑，油漆早已褪得发灰。减震钢板多半早断了，在 107 国道的破损路段，整个车厢剧烈颠簸，车窗哐啷作响，感觉随时会散架。座位早已坐满，过道里也挤满了人，就坐在行李上。空气里混杂着红梅烟的焦苦味、汗酸味，还有几双脱了鞋的脚散出的酸腐气息。

车开到韶关坪石附近时，天已经完全黑了。南岭的山一层压着一层，只有车灯劈开前方一小段灰白的路面。忽然“哐当”一声，车猛地停住。车门打开，冷风带着山里的湿气猛灌进来，也带上来一个人。

车顶那盏昏黄的小灯泡随着发动机的怠速忽明忽暗。借着那点光，我看见她二十出头，个子不高，身形单薄。她穿着一件米色束腰风衣，那是当时城里年轻女人常穿的样式，腰带松松系着，里面是一件白衬衫，下身是一条熨得笔直的直筒裤，脚上是双沾了灰的浅口皮鞋。

她明显走了很远的路，头发被夜风吹得散乱了，脸上还能看出精心化过妆的痕迹，细细的眉毛，淡淡的口红，只是嘴唇已经干裂。

在这满车灰扑扑的中山装、旧迷彩服和沾满水泥灰的军大衣之间，她显得格外突兀。

“到哪？”司机不耐烦地问。

她把手伸进风衣内侧的口袋，摸了很久，才掏出一张皱巴巴的纸币，小声说：“到前面镇上。”

司机撕下一小截票，没再说什么。

车厢已经没有空位。她只能站在车头发动机盖旁边。那里是整辆车最难受的地方。铁皮滚烫，柴油机的震动透过鞋底直往骨头里钻，噪声大得连说话都要靠喊。

她抱着一个洗得发白的帆布包，两只手用力握在一起，像怕被人抢走。

“哎，你看。”发小黑皮用胳膊肘捅了我一下，压低声音笑着说，“城里妹子。”

他嘴里叼着半截红梅，烟灰抖在裤子上也不在意。旁边坐在蛇皮袋上的二愣子抬起头，顺着他的目光看过去，也咧嘴笑了。

工地上干了一年活，除了灰尘就是骂声，连女人的影子都难见一次。所有的欲望和憋闷，都被压在日复一日的搬砖和搅水泥里，一旦出现一点不同的气息，就像闷罐里的气突然找到了出口。

黑皮踢了我一脚说：“你喊她一声。”

我本来没想动，可周围几个人都看着我，像等着看个热闹。我不想显得自己扫兴，也不想被人说没种。脑子一热，我把腿往过道一伸，冲她喊道：“妹子，没座吧？站着多累，来这边挤挤，哥这腿暖和，给你当凳子。”

话音刚落，周围立刻响起一阵哄笑。黑皮吹了声口哨：“过来嘛，别害羞呀。”

二愣子拍着蛇皮袋说：“我给你让个地方。”

她没有动，甚至连头都没回。只是抱紧帆布包，站在原地，目光越过车门窗，盯着外面无边的黑夜。冷风从车门缝里灌进来，吹动她额前的头发。她的侧脸在忽明忽暗的灯光里显得那样清冷。

我们的笑声渐渐稀落下来。这种无声的拒绝，比骂人更让人难堪。黑皮脸上的笑意收紧，猛地推了我一把：“去啊。”

我踉跄着走到她面前。她抬起头，那张年轻而疲惫的脸，透着掩饰不住的清秀。我心里忽然升起一股莫名的躁动。伸出手，在她脸颊上轻轻捏了一下。

“啊！”她猛地缩向车门，像被火烫到一样，声音细得发颤：“别碰我……”

那一瞬间，我看清了她的眼睛。恐惧，屈辱，还有一种拼命压抑的绝望。那眼神，像极了我远在老家、那个胆小又内向的妹妹。我心里猛地一沉，像从高处踩空了。刚才那点酒意和起哄带来的兴奋，瞬间消失得无影无踪。

黑皮站了起来，大喊道：“装什么清高？穿成这样，不就是出来卖的？”

车厢里哄笑声更大了。我却低下头，退回座位，假装去整理脚边的编织袋，像个逃兵一样避开所有人的目光。正当我以为她也会顺从地瘫软下去、任由这满车厢的恶意继续欺凌时，她猛地甩了甩凌乱的长发，像一头被逼到悬崖边的小兽，用尽全身力气发出一声凄厉的嘶吼：“停车！我要下车！”声音嘶哑，却像刀子一样锋利。

司机从后视镜看了一眼，骂了一句：“还没到站！”

“我要下车！”她又喊了一遍。

车终于在盘山路边停下。车门刚打开，她就跳了下去。车灯照亮了一小片荒草地，旁边是一间废弃的修车棚，铁皮屋顶锈得发黑。再远，就是深不见底的山谷。她没有回头。米色的风衣在夜色里晃了一下，很快被黑暗吞没。

车门关上。客车重新启动，继续向北开去。

黑皮靠在椅背上，很快打起了呼噜。二愣子也缩着脖子睡着了。仿佛什么都没发生。

只有我睡不着。窗外是无边的黑夜，偶尔掠过几棵树的影子。我忽然想起在老家送我出来时，我妹妹站在村口，也是这样瘦小。我也被人骂过乡巴佬，被包工头拖欠过工钱，被城里人用那种无形的轻蔑推开过。可就在刚才，我却成了自己最痛恨的那种人。

我闭上眼睛，却还是看见她在车头的样子。
