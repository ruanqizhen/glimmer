# 兵法

   *1998* 

我是高大勇，双河村的村长。八十年代末退伍后，我跟着浪潮去广东闯荡了几年，攒了点辛苦钱，九十年代初回到老家，被乡亲们推举当上了村长。回村后，我把当年的老战友周强安顿在村东头，让他帮我看守那口大鱼塘。

那时候，为了把村办塑料制品厂办起来，我几乎天天都要陪县里来的客人到鱼塘钓鱼。人在场时，我总是满脸堆笑，中华烟一根接一根地递，好茶好水伺候得周周到到，显得格外殷勤。可客人前脚一走，我回到屋里关上门就忍不住骂娘。这当村长的，比部队里复杂多了，哪路神仙没拜到位，这厂子的公章就别想盖下来。我最头疼的是县工业局的钱科长，卡得最死，迟迟不肯松口，为这事我真是绞尽了脑汁。

1995年那个下午，我开着突突作响的手扶拖拉机，拉了一张大拉网，叫上几个村民在村东鱼塘里折腾了大半天，又轰又赶又拉网，结果一条成鱼都没捞上来。临走时，我反复叮嘱周强，明天县工业局的钱科长要带人来钓鱼，你把鱼棚锁死，不管塘里发生啥事，都别出来掺和。

第二天九点多，三辆黑色的桑塔纳轿车卷着一路黄尘，浩浩荡荡开进了村东鱼塘。我那天故意称病没亲自陪同。后来周强告诉我，看得出其中有几个已经是熟客了，一下车就领着其他人，在鱼塘边一字排开，撑遮阳伞、开鱼饵盒、上蚯蚓和酒米、甩出玻璃钢钓竿，很快就进入了状态。

可那些鱼前一天刚被我带人吓怕了，全缩在泥底，哪敢出来咬钩？差不多一顿饭的功夫，水面浮标纹丝不动。有几个性急的家伙沉不住气了，四下张望，瞅见旁边村西那口老鱼塘，犹豫一阵，就扛着鱼竿转移了过去。不久，其他七个人也被那边的惊叫声和水花声吸引，纷纷跟了过去，那鱼塘边顿时响起一片欢声笑语。

正当钓客们钓得忘乎所以时，鱼塘边不知啥时候多出一个六十多岁的老头，只见他双手叉腰，布满老茧的脸涨得通红，破口大骂，好啊，你们这帮强盗，我说我这塘里的鱼怎么年年亏本，原来青天白日就有人敢来明抢，都给我站起来。

众人赶紧站起来，面面相觑。其中那个微胖的钱科长赔着笑脸说，老伯，我们以为这是村上的公塘，不好意思，钓的鱼我们按市价付钱。

老头可不是好说话的主，拐杖往地上一跺，抓到了才付钱？我的鱼在水里活蹦乱跳时可是值大价钱的，要想走人，把你们的车都留下。

钱科长有些急了，说老伯息怒，我们是双河村村长高大勇请来的。

老头火气更大，吐沫星子乱飞，村长算个屁，告诉你，老子活到这把岁数天不怕地不怕，专治你们这些当官的。

正吵得不可开交时，我腰上的BP机疯狂响了起来。收到周强的传呼，我赶紧骑上那辆嘉陵摩托车，一路油门轰到底赶到现场。钱科长一见我，如获救星，拉着我耳语了几句。我就把老头拉到一边，小声解释情况。没想到老头一听又跳起来，大声嚷嚷着，原来你们官官相护，车牌号我都死死记下了，我要告到中央去。

钱科长把我拉到一旁，擦着冷汗问这老头是什么来头。我一脸无奈，压低声音说，钱科长，实不相瞒，这老家伙是我们村头号难缠的告状专业户，我的前任就是硬生生让他给告下台的。钱科长脸上的肌肉微微颤动了一下。

我又咬咬牙说，这老东西太过分了，钱科长您放心，我豁出这个村长不当，带两个后生把他绑回去，也绝不让他耽误各位领导的前途。

钱科长连连摆手，大勇，你千万别硬碰硬，对这种基层群众要做细致的思想工作，不能激化矛盾。这事就全权交给你了，一定要安抚好，至于你们村办厂的困难，我们县里尽快帮着解决。

第二天，办塑料制品厂的所有审批手续果然一路绿灯，全都批了下来。

几天后的一个傍晚，双河村村口的聚贤茶楼里，我给对面的老头倒上一杯热茶，双手举杯碰了一下。老头哪还有半点那天泼妇骂街的蛮横样？笑眯眯地和我碰杯。他其实根本不是什么告状专业户，他叫郭本旺，是我的前任老村长，也是当年最支持我回村发展的领路人。
