# 高棋

一九九八年的那个夏天，雨像断了线的珠子，没完没了地砸在窗玻璃上，也砸在每个人的心坎里。整个城市都泡在水里，报纸上天天登着灾区的惨状：断壁残垣，泥汤子一样的街道。机关里头，大家都猫着腰过日子，等着市里那场机构大调整的风向。谁知道哪天就轮到自己下岗？空气里一股子霉味儿，夹杂着烟丝和等机会的酸溜溜劲儿。

我在局里供销科熬了十几年，日子过得像钟摆一样准时：早上七点半打卡，下午五点半下班。老婆在家念叨着单位老王的媳妇儿用上了进口的“松下”电饭锅，煮出来的米饭晶莹剔透，不粘锅底。她天天在我耳边嗡嗡：“守志，你也给咱家弄一个吧，省得我老守着煤气灶。”我攒了半年，终于兜里揣着六百块——六张崭新的“大团结”。下班后直奔百货大楼，提回家，晚上就能闻着米香味儿美滋滋地吃一顿了。

谁知那天下午，办公室里闷得像蒸笼。我于是溜达到有风扇的会议室里去看电视，新闻里一个白头发老太太，坐在半淹的灶台边，死死攥着个糊满泥的铝锅，眼里全是绝望。忽然就想起了老家土灶前忙活一辈子的老娘，心头一热。

就在这时候，门“咣当”一声开了。工会的刘大姐抱着个糊红纸的捐款箱进来，开口处像张血盆大口。小王跟在后头，举着“海鸥”相机，随时准备抓拍“先进事迹”。

我那天真是鬼迷心窍，脑子里全是那老太太和她的破锅，心想：没锅，煮啥米饭？一咬牙，我猛地站起来，在大伙儿瞪圆眼的注视下，从兜里掏出那卷钞票，直接塞进箱子。

走廊里，几个副局长正在聚堆儿抽烟，低声嘀咕。他们都还没有捐款，局长没露面，谁也不敢先捐。大家都在等局长定调子。

“焦守志，六百！”刘大姐喊出这一声，嗓子都走调了。相机“咔嚓”一闪，晃得我眼冒金星。走廊里那几个副局长，烟头齐刷刷停在半空，看我的眼神像见了外星人，又像在看个自找死路的疯子。

下午，大红榜贴出来了。我的名字像尊大佛似的压在榜首，“陆佰元”金光闪闪，刺得全楼人心慌。下面才是局长三百，副局长两百，往下递减……这种视觉上的“僭越”，在九十年代的机关里，简直就是自杀。有人私下嘀咕：“焦守志这是闹哪出？想给局长难堪，还是觉得自己比局长更有觉悟？”

大家等着看我倒霉。平日里走得近的哥们儿，看我的眼神带上了怜悯，像在看个即将被晾起来的边缘人。茶水间里，议论声起伏：“这小子完了，非要抢局长的风头。”

没想到，半个月后，我竟然被提拔成了供销科科长。全局上下，眼镜碎了一地。办公室里先是死一般的安静，然后大家伙儿瞬间变脸。

“焦科长，高，实在是高！”被调去人事部门的老科长特意过来给我点烟，压低嗓门，一脸佩服，“当时大伙儿都以为你犯傻，谁知道你是摸准了上头的风向？这六百块砸得准啊，早知道市里要树典型了吧？”

茶水间里，有人冷笑：“瞧瞧人家焦守志，平时装老实，关键时候真阴损。这哪是捐款？这是当众给局长下套呢！心思深着呢，咱们学不来。”

我坐在新办公桌前，看着那张任命状，心里全是苦笑。我没法告诉他们，我那天真就只是想买个电饭锅。辩解过一次，对方眯眼一笑：“懂，哥们儿都懂。”

在这个聪明人扎堆儿的围城里，既然大家都认定我老谋深算、有背景，恐怕我也只好戴上这副面具，孤独地演下去了。
