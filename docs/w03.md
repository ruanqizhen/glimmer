# 夜里的黄包车

工厂的大门关上得很干脆，没有太多解释，只是一纸通知，我们这些人就被从原来的生活里挪了出去。我在那之后去拉了黄包车。

车是租的，深紫色的布棚，铁架子很沉。每天凌晨四五点，我得把车从胡同里推出来，动作要轻，怕吵醒邻居。那时住的还是筒子楼，墙薄，人多，谁家都不宽裕。

拉车这行，有制度。交通局规定，单号车跑单日，双号车跑双日。官方说法是“公平”，让大家都有饭吃。但现实是，肚子不认这个规矩。老婆要买菜，孩子在学校要交杂费，单双日并不能帮你多撑一天。

轮到我不能出车的日子，只能等天黑。等那些人下班了，再把车推出来。夜里的生意不好，有时候一晚上拉不了两趟，还得随时留意巡查。一旦被逮住，罚一百块。

我记得很清楚，有一天夜里，我在路边等活，来了一家五口。两个大人，三个孩子。他们看了一眼车，又看了我一眼，什么都没多说，就开始往上挤。孩子蹲在踏板上，大人叉着腿坐着，一车人刚好装满。我心里清楚，这不合规矩，也不安全，但我没吭声。

那种时候，你很难拒绝。你知道这是超载，但你也知道，这可能是今晚唯一的一单。

起步的时候，我几乎没把车拉动。腰一下就塌了，背弯下来，手臂绷得发硬。轮子在地上晃了几晃才动起来。路边有人看，有人避开视线。我当时只想着一件事：别停，停下来就再也走不动了。

之后我又拉了一对兄弟。他们说话很客气，问我一个月能不能挣到五百块。我没正面回答。五百块这个数，在不生病、不被罚、不下雨、不下雪、每天都有客的情况下还是有可能达到的。我也没说自己以前在厂里上班，我怕提到“下岗”这个词。

他们下车时，连找零都没要就走了。我在后面说了好几声谢谢。我知道，那不是因为车拉得好，是因为他觉得我可怜。

但是，同情这种东西，对我们这样的人来说，并不总是好事。你们要是因为觉得“不忍心”“不好意思”而不坐车，对我来说，反而更难。我们卖的是力气，不是情绪。你不坐，我就没有这一单。

明天又轮到我出车。我得早点起，在天亮之前，把车推出来。
