# 民告官告赢了

这个案子在我们那边挺出名的，因为是少有的民告官还告赢了的案例。里面很多细节我都是事后看报纸才知道的，这里给大家讲讲这事儿的来龙去脉。

故事得从 1995 年春天说起。那会儿正是农忙时节，刘凤海、刘凤岐哥俩从姐姐家借了 500 块钱，这在当时可不是小数目。他们琢磨着买麦种，把家里那70亩地好好侍弄侍弄，指望秋后能翻个身。4 月 16 号一早，哥俩兴冲冲挤上了从县城回永兴村的公共汽车。车上挤得慌，半路上来个胖售票员，张口就说他们少买了一块钱车票。弟弟刘凤岐脾气直，当场就嚷嚷起来：“明明前一个售票员都说好了，你凭啥变卦？”两人就在车上推搡起来。

下车的时候，哥哥刘凤海还埋怨弟弟："就为一块钱，至于吗？快摸摸咱那钱还在不在。"这一摸，把哥俩的魂儿都吓没了——怀里那 497 块钱，没了！

两人立马反应过来，肯定是那胖子趁乱偷的。他们发了疯似的拦车追上去，那胖子先是死不承认，后来心虚了，想私了给点钱就算了，结果三个人扭打成一团。正好河南乡派出所的民警杨玉峰和沈志龙路过，把他们仨都带回了所里。

刚下车，那胖子偷偷把钱扔到车轱辘底下，没想到被眼尖的刘凤岐看见了。把钱捡起来一数，497 块，分文不差。人赃俱获，眼看这事儿就要了结了。没想到那胖子亮出了身份证——克山县公安局交警队民警，于忠滨。

民警杨玉峰算是个正直人，虽然觉得警察做这种事丢人，但钱找回来了，他就把于忠滨放了。刘家哥俩也是本分农民，寻思着民不与官斗，拿回钱赶紧回家种地要紧。谁知道，事情这才刚开头。

于忠滨回去后不但不反省，反而恶人先告状，跟局长田晓明汇报说自己根本没偷钱，是刘家兄弟打他、诬陷他。田晓明这个局长也不调查核实，就信了这一面之词，直接定性说刘家兄弟是“车匪路霸”，还亲自带人去村里抓人。就连秉公办事的杨玉峰和沈志龙两个民警，也被他停了职。

当天下午，哥俩就被抓进了公安局。审讯的时候，他们一遍遍喊“我们是丢钱的受害者”，换来的却是一顿顿毒打。听说有警察拿烟头烫刘凤岐，还嘲笑说“烫你算便宜你了”。刘凤岐被关了一个多月，家里东拼西凑又凑了 500 块“赎人钱”，才把他放出来。

可哥哥刘凤海，那个能扛几百斤麻袋的壮汉，再也没能活着出来。8 月 16 号，刘家接到通知说刘凤海“病了”。刘凤岐赶到公安局，田晓明拿出一张纸让他签字：“你哥打了警察，现在私了吧，签字领人，再赔点钱。”刘凤岐说什么也要先见人。他被带进一间黑屋子，在黑暗中摸到的，是哥哥冰凉的尸体——瘦得只剩皮包骨头，哪还有半点壮汉的模样。

公安局说人是“病死”的，什么病、在哪治的、怎么死的，一概不说。更让人气愤的是，不等家属同意，也不查明死因，尸体就被强行火化了。村里都在传，刘凤海是被超期关押、虐待致死的。那年他才 36 岁，还没娶媳妇，一直帮着弟弟妹妹过日子。

哥哥死了，连骨灰都没见着，刘家人咽不下这口气。从那年起，他们就开始了漫长的上访申冤路。县里、市里、省里，最后跑到北京。刘凤岐和四哥刘凤金，风里来雨里去，睡过火车站，住过桥洞，有时候好几天吃不上一口饭。

村里人劝他们：“胳膊拧不过大腿，算了吧。”可他们就认一个死理：总得有个说理的地方！就这么一股倔劲儿，坚持了整整五年。家里穷得叮当响，债台高筑，刘凤金后来都被逼得搬离了村子。

终于，这事惊动了省里高层。省人大副主任单荣范批示彻查此案。齐齐哈尔市公安局重新调查，真相大白——刘家兄弟是受害者，却被当罪犯对待；偷钱的于忠滨逍遥法外，无辜的刘凤海反而被关押致死。这从头到尾就是一桩冤案。

1997年6月，于忠滨因诬告陷害罪被抓。2000年1月，田晓明因涉嫌玩忽职守被齐齐哈尔市检察院逮捕，4月开庭审理。村里当时都轰动了，报纸上的标题是“农民告倒公安局长”。可结果呢？5月判决下来，田晓明虽然被判玩忽职守罪成立，但“鉴于他对公安工作的贡献”，免予刑事处罚。就这么不痛不痒地放了。其他涉案的李广才、李长有、李文有几个人，虽然也涉嫌徇私舞弊，最后也都不了了之。

那年除夕，刘家人在街头给刘凤海烧纸，总算能说一句“哥，我们给你伸冤了”。可这冤伸得值吗？一条人命，五年的血泪，换来的不过是个“免予处罚”。
