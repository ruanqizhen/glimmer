# 膝盖

我十五岁那年，村子忽然被一种躁动的红色淹没了。

那是盛夏的一个正午，太阳毒得能把老槐树的树皮烤裂。我的小学老师张先生，被人从他那间阴暗的草房里拽了出来。张先生是村里唯一一个在城里念过书的人，十分讲究礼貌，见谁都先点个头。平日里说话轻声细气，干活再累也不吭声。可那天，他成了“阶级敌人”。

“跪下！”

队里赶大车的王二嘴里带着一身烟酒味儿吼了一嗓子，顺手把从张先生家里翻出来的几只细瓷碗往石阶上一摔。瓷碗碎裂的声音清脆又突兀，青白色的碎片撒了一地，在阳光下闪着冷光。

“向伟大领袖认罪！臭老九！”

张先生一句话也没说。他身子晃了晃，像被风吹折的枯木一样，在那堆碎瓷片上跪了下去。

我站在离他不远的地方。那一刻，我清楚地听见瓷片扎进皮肉的声音，低而闷，随即是骨头压在硬物上的摩擦声。一缕细细的血顺着石阶流下来，在干裂的泥土上洇出一片暗色。他额头上的汗珠往下滴，混着血腥味，在热浪里发酵。

我混在人群里，膝盖一阵阵发软，想往后退，却没敢动。我偷眼看周围的大人，有的把头埋得很低，有的眼神空洞，冷得让人心里发慌。

这时，我的同学李虎子冲了出来。虎子平时总被人欺负，可那天却像换了个人似的。他扑上去，对着张先生的脸左右开弓，耳光声响得刺耳。他喊得比谁都卖力，像是在向全村宣告：打得越狠，就越安全。

那一刻，我只剩下一个念头 —— 我绝不能跪在那里。

打那以后，我成了村里最积极的人。上面一有新精神，我就抢着冲到村口的大喇叭前念文件；队里开批判会，我通宵达旦地写发言稿。我并不认识那些远在京城的“大人物”，可我还是用自己能想到的最恶毒的词句去咒骂他们。

站在土台上领读时，我的声音总是最大的，喊到嗓子破音也不停，我得让人听见我的“觉悟”。只有我比别人更愤怒、更坚定，别人才会觉得我是“自己人”，我才不用担心哪天会轮到我的膝盖受罚。

那些年，我像个被恐惧驱动的木偶。每天醒来先对口径，出门先看风向，说话前先掂量分量。

后来，碎瓷片被扫走了，石阶上的血迹也被雨水冲干净了。可我始终觉得，那道血印并没有消失，它留在了我的身体里。

直到今天，只要听见稍大一点的动静，我都会下意识地弯下腰去。
