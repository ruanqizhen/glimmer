# 清零

多少年来，咱机床厂财务科最响亮的声音都不是人说话，而是我手里那把老紫檀木算盘的“噼啪”声。

早年间，我有幸遇到一位供销社退下来的老前辈，拜他为师，学了一手正宗的“百子盘”基本功和“狮子滚绣球”的连环指法。我是市里珠算大比武的常年冠军。在财务科这方天地里只要我桌上那清脆的珠子声一响，科里其他人的工作节奏就不自觉地跟着我走。哪怕后来厂里给每个人发了那种带个小太阳能板的卡西欧计算器，我也没拿正眼瞧过。

“那玩意儿按快了容易漏键，没有手感。”我总是戴着蓝布套袖，端着搪瓷茶缸，这么教导我的几个徒弟，“算盘是有灵性的，手指头拨到珠子，听到那声脆响，账才算得踏实。”

年底扎帐的时候，是最能体现我老林价值的时刻。桌上的牛皮纸凭证堆得像小山，几台计算器按得“滴滴”乱叫，年轻人们经常因为中间错按了一个键就得全部推翻重来。而我，左手飞速翻动账本，右手三指如飞，“破头乘”、“九归除”，一招一式行云流水。清脆的木珠碰撞声里，一套几百万的明细账被我理得清清楚楚，分毫不差。厂长见了我，都得客客气气地递根红塔山，喊一声“林师傅”。

那是我引以为傲的江湖，而我是这个江湖里稳坐钓鱼台的绝顶高手。

然而，我的噩梦还是来了。年初的时候，厂里开始搞“现代化企业管理试点”，搬来了几台笨重的“长城”微机。新来的大学生小陈被任命为信息主管。他戴着副厚底眼镜，不拿算盘，也不拿账本，每天就拿着几张四四方方的黑色塑料盘在科里穿梭。

“林师傅，以后咱们的账目都要录入数据库了，用的是 FoxBASE。”小陈在我的办公桌旁比划着，“以后查账，只要敲一下回车，所有的汇总数据一秒钟就能调出来。”

我听得一头雾水：“什么贝斯？啥数据库？库房在哪？一秒钟调出来，它不用一笔一笔地加？”

为了证明他的话，小陈把我拉到了电脑前。跟我说这是 UCDOS，上面跳动着刺眼的绿色光标。他让我报一个月前的某批次钢材入库明细。我转身去搬那摞积满灰尘的厚账本，刚翻到一半，就听见旁边那台宽大的打印机发出了“吱啦吱啦”的刺耳尖叫。

不到两分钟，一张连绵不绝的带孔打印纸被吐了出来。上面整整齐齐地列着所有的明细、单价、总额，甚至连材料损耗率都自动算好并排好了版。

我彻底傻眼了。

我不信邪，一把扯下那张纸，把我的紫檀木算盘拉到面前，开始一笔一笔地复核。我的大拇指和食指上下翻飞，额头上渗出了细密的汗珠。半个小时后，我看着算盘上最后停留的数字，手开始不受控制地发抖。

分毫不差。

我引以为傲的“狮子滚绣球”，我练了三十年、磨出了厚厚老茧的指法，在这个甚至没有实体拨珠过程的“微机”面前，变成了一个迟缓、笨重且毫无意义的笑话。

不到半年时间，科长就找我谈话了。“老林啊，财务系统全上了微机，现在讲究‘会计电算化’。”他眼神闪躲，不敢看我桌上的算盘，“你连五笔字型都背不下来，键盘也敲不利索。厂里给你安排了新任务，调到后勤库房，你的记账底子还有用武之地。干啥都是为了厂子做贡献嘛。”

我没有闹，也没有求情。那天下午，我默默地把那把紫檀木算盘，用一块干净的红布仔细包好。走出财务科的时候，里面没有了清脆的算盘声，只有键盘那种沉闷、单调的“哒哒哒”声，像是一台没有感情的机器在一点点咀嚼掉我的大半生。

在仓库里坐了半个月冷板凳后，我觉得我不能就这么废了。电脑是快，但老祖宗传下来的智慧不能断。算盘里的加减乘除，藏着规矩，藏着方圆。师傅当年传授我技艺的时候，就一个要求，不能让这门手艺在咱们这代人手里断了根。我决定去街头摆个摊，不为赚钱，就为了让人看看这门绝活。厂里不用了，我就不信，偌大个城市，没人识货了。

我请厂里木工班的老刘给我打了一把一米多长的大号教学算盘，刷上黄黑相间的漆。又去五金店买了个便携式扩音器。下班后，我在市文化宫附近找了块空地，把大算盘往街边墙上一挂，打开麦克风开始吆喝。

“大家停下步子看一看，算盘不仅能算十进制，还能算咱们老秤的十六进制！一两等于零点零六二五斤，算盘口诀叫‘一退六二五’，快得很呢！” “我再给大家演示个‘九归诀’……”

路上人来人往，多数人夹着公文包看都不看就匆匆走过，偶尔几个看热闹的停下来，盯着算盘叹了口气，摇摇头又走了。

终于，几个穿着校服、背着双肩包的中学生叽叽喳喳走过来，看见我的大算盘来了兴趣。我顿时来了精神，一招“凑倍乘”，双手齐下，演示了一个 12345 乘以 11111。

其中一个学生掏出了计算器，过了几秒钟才惊讶地说：“哎？还真对啊！”

另一个学生也凑过来：“这么神奇，大爷，这难学吗？”

结果先前那个学生一撇嘴：“对是对，但有啥用啊？大爷，您这算盘能算根号 37 吗？”

我愣了一下。开平方？我当然能算！珠算里有“减奇数法”，也有“少广术”。 

我深吸一口气，刚把手腕悬在半空，布数定位。

“哎呀，算不出的，走吧走吧。”那个学生大拇指一按，“6.08276！走啦去打台球！”

几个孩子立刻随声附和，哄笑着跑远了。我悬在半空的手，僵住了。是啊，我能算，可算得再精妙，在他们眼里又有什么意义呢？

秋风吹过，卷起一地的杨树叶。街上每个人都在急匆匆地赶路，再也没有人需要用这四方木框里的珠子去丈量生活了。

我取下算盘，伸出右手食指，沿着中间的横梁，从右到左，稳稳地一扫而过。

“啪。”

所有的算盘珠子瞬间归位，这是珠算里每次运算前必须做的第一个动作，叫“抹盘清零”。我把算盘装进布袋里。回家吧，我对自己说。在这个狂奔的新时代里，我已经被清零了。
