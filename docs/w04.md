# 准生证

一个新生命的诞生，并不完全取决于生理条件，而更像一项需要层层报备、逐级盖章的工程。

一切要从结婚说起。

最初的资格证明是结婚证。那是一个工作日的下午，我和未婚妻本该在单位上班，却请了假，在所在辖区的街巷里来回穿行。民政局藏得很深，像是故意回避我们。问了不少路人，得到的回答五花八门。最终，我们还是在一条正在施工的路段的尽头找到了那个地方。接下来是排长队，交了一摞说不清名目的费用。

快门响起的时候，我们都没什么表情。红本子到手，再一看照片：头发凌乱，脸色灰暗，像是刚从工地下来。这就是我们合法关系的第一份证明，来得并不浪漫，却异常正式。

但这张证件只是表示“可以结婚”，并不意味着“可以生育”。

接下来，是一套被称为“学习”的流程。街道统一组织新婚夫妇参加计生培训，内容大多围绕“基本国策”和“优生优育”。教室昏暗，讲台上的老师语气郑重，台下的人心不在焉。课程结束时，每人领到一袋避孕用品，胶皮味很重。这套逻辑清晰而自洽：在你有资格生育之前，必须先学会如何不生。

真正的难关，是拿到“指标”。

我们在单位、街道和居委会之间来回跑，填表、签字、盖章。计划生育并不只是家庭事务，它有完整的档案体系。居委会的计生干部对我们的生活了如指掌：婚龄、住址、工作单位，甚至例假时间，都被记录在表格里。已婚未育的女性需要定期检查，以证明一切仍在“计划之中”。那是一种持续的确认过程，身体本身成了需要随时接受抽检的物品。

终于，在反复核对之后，我们拿到了一张纸质粗糙的证明。纸张发黄，格式简陋，却盖着鲜红的公章。这张纸，意味着我们可以“开始尝试”了。

但事情还没完。

按照流程，还要去计生办。在国徽下面，郑重承诺"只生一个"。这不是口头表态，而是白纸黑字的承诺。协议里写得很清楚：孩子出生后，必须按规定落实节育措施，否则后果自负。为了这一纸协议，我们又交了一笔费用，名目叫"保证金"或"押金"，具体保障什么，没人解释。

最终，我们从街道计生办的柜台前领取了那张真正意义上的通行证——准生证。

直到那一刻，紧绷的神经才稍稍松动。我们终于可以去医院建档，办理母子健康手册，堂堂正正地进入产检通道。接下来，只需小心翼翼地等待那个日子的到来。
