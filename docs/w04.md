# 准生证

一个新生命的诞生，并不只是两个人的决定。它更像一项需要层层确认、逐级签字的工程，嵌入在计划生育这一基本国策之中。生育从来不是纯粹的家庭私事，而是制度框架内的一环。

回望那段经历，我常觉得自己走过了一条漫长而曲折的路径——每一步都伴随着表格、印章和等待。下面，就从结婚说起。

按照婚姻法规定，男满22周岁、女满20周岁即可登记结婚。但在当时的政策语境里，晚婚更被提倡——男25周岁、女23周岁以上。晚婚不仅是一种倡导，还与假期、福利等现实利益挂钩。

那是一个工作日的下午。我和未婚妻请了假，在辖区街巷里寻找民政局。它藏在一条正在施工的小路尽头，我们问了好几拨人，才终于找到。登记并不复杂，却也不轻松：户口本、身份证、单位证明、照片一样不能少；排队、填表、缴费，一项接一项。半天时间过去，我们终于拿到那本红色的结婚证。

我们快就意识到，结婚证只是起点。它意味着我们的婚姻关系正式进入基层管理视野——居委会、单位都会开始记录我们的婚育状况。至于生孩子，那是下一道关卡。

新婚后不久，街道通知我们参加计划生育培训。那是一种例行程序。内容围绕政策宣传、优生优育知识、避孕方法等展开。教室光线昏暗，讲台上的老师语气郑重，台下的人心思各异。

讲课强调晚婚晚育的好处、遗传病筛查的重要性，也讲解常见的避孕方式。课程结束时，每人发一袋药具和宣传材料。胶皮味混着油墨味，令人尴尬又无奈。

逻辑其实很清楚：在被允许生育之前，要先学会如何不生育。从公共卫生角度看，这一步确实提高了不少人的健康意识；但从个人感受上说，那种被规训的意味，也让人心里压抑。

婚后未育的夫妻会被纳入计生档案。居委会或单位计生专干会定期了解情况，更新住址、工作单位等信息。有些地方还会要求女方做例行检查，以确认未怀孕或已采取节育措施。

那是一种持续的“确认”。身体仿佛成为需要随时被核验的对象。我的妻子去过几次街道卫生站做检查，过程谈不上粗暴，却始终让人不自在。若配合不好，难免在评优评奖、福利待遇上受影响。对我们而言，这意味着私人生活不断被公共权力触碰。

真正的难关，是申请生育许可，也就是俗称的“准生证”。

流程通常从女方户籍所在地开始。我们在单位、居委会和街道办之间来回奔波，填表、签字、盖章。需要提交结婚证、户口本、身份证、单位或居委会开具的婚育证明。基层会核实是否初婚初育、是否达到晚育年龄、是否有违规记录。

政策执行在城乡之间有所差异：城市普遍是一对夫妇生育一个孩子；农村在符合条件时可申请生育第二个，但程序更为繁琐。具体审批层级因地区而异，有的只需街道审核，有的还要报乡镇或区县部门备案。没有统一的时限标准，一切都取决于当地节奏。

那段时间，我们填过的表格多到记不清名字。每一页都需要红章确认。章盖得越多，纸越显得沉重。

终于，经过反复核对，我们拿到一张纸质粗糙的证明。纸张发黄，格式简单，却印着鲜红的公章。那一刻，我们才算获得“开始尝试”的资格。

不少地方在发放准生证前，还会要求签订计划生育协议。我们也签了。在国徽图案下，郑重承诺按政策生育，并在产后落实节育措施。

协议并非象征性的。很多地方会收取一笔保证金，数额不算巨大，却足以让人记住这份约束。等落实节育措施后，方可退还。那是一种制度化的提醒：承诺不仅是口头的。

上环在当时是最常见的选择，成为许多家庭的“标准答案”。

最终，我们在街道计生办窗口领到那张真正意义上的通行证——准生证，也叫生育服务证。它印着国徽，有效期通常为一两年，标明允许生育一个孩子。没有准生证，孩子出生后的上户口、入托入学都会遇到麻烦。它不仅是一纸许可，更像是一道通往未来的门票。

直到那一刻，紧绷的神经才稍稍松动。我们可以去医院建档，办理孕产保健手册，正式进入产检流程。
