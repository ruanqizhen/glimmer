# 深圳被收容经历

1997 年 5 月的一个上午，我当时还在办公室上班，原本挺忙的。那天两件事赶在一块了：一是我的未婚妻要到深圳，二是我一个新疆的同学路过，要在火车站见一面。我跟单位打了个招呼，拿了三百块钱，就急急忙忙奔了火车站。

坏就坏在太急了，证件全落在了办公桌上。

刚到火车站，在港龙大酒店附近，一个保安就把我给揪住了，张口就要证件。我当时还挺客气的，跟人家解释：我是哪个单位的，出来接人，忘带了，离得不远，能不能通融一下？可人家根本不听，看我的眼神像盯着个猎物，直接把我推到了警察面前。

我以为警察能讲点理，毕竟我穿着体面，谈吐也不像坏人。我提议去旁边的电话亭给单位打个电话，这事儿不就清楚了吗？可那警察二话没说，使劲一拽，就把我塞进了一辆闷得要命的破车里。车里坐得密密麻麻，什么样的人都有，大家都低着头，气氛特别压抑。

接下来的两天，像是一场荒唐的梦。我们先是被拉到罗湖口岸派出所，六平米的小屋子挤了五十多人，连个转身的地方都没有。晚上又被拉到了银湖。

在那儿，有个常进收容所的哥们儿偷偷提醒我：“有钱快藏起来，一会儿要搜身。要是钱够，没准儿能把自己赎出去。”我赶紧把那三百块钱卷成个小卷，硬塞进了头发里。果然，警察搜了一圈，没发现我头上的“玄机”。

当时有个规矩：交 600 块钱能放人。我只有 300 块，不够数。第二天，我们又被拉到了东莞。到了那儿，价格降到了 400 块。我当时心里那个恨啊，心想出门怎么就没多带一百块钱呢？

绝望之下，我把单位的电话告诉了一个凑够钱准备走的人，求他出去后帮我打个电话。剩下的时间，我只能在发臭的监仓里待着，看着身边各种各样的人，有小偷，也有像我一样倒霉的游客和商人。我当时就在想，这些执法的人，怎么能这么对待普通百姓呢？

到了第三天，奇迹发生了。一个警察叫我的名字，说有人来接我了。

走出收容所大门，第一眼看到的就是我未婚妻。她整个人都憔悴得不像样了，眼圈黑得吓人。后来才知道，我失踪这两天，他们全疯了，报了案，查遍了全市的医院，甚至连无人认领的尸体都去核实了。谁能想到，一个大活人，竟然是被关进了收容所里。
