# 缝补余生

我最后看了一眼妮儿，在心里轻声跟她说了一句：“妮儿别怕，妈很快就来陪你了。”

从火葬场旁边的骨灰堂出来，天已经有点阴了。我摸了摸兜里，只剩下几张被汗水浸得发软的毛票。为了给妮儿治病，能卖的东西早就卖光了，连那间勉强能叫“家”的小屋，最后也被房东收走了。

八岁。她才八岁。

那一刻我很清楚，我这一辈子，大概也就到这儿了。

天快黑的时候，我在路边拦下了一辆红色的夏利出租车。

车里暖风开得很足，一股子说不清的味道——劣质烟草、机油，还有一点汽油的酸气，混在一起。司机是个中年男人，穿着件旧皮夹克，袖口油亮油亮的。

“去哪儿？”他没回头，顺嘴口了一句。

“北桥底下。”

他说话的声音顿了一下，从后视镜里看了我一眼：“大姐，那地方是烂泥滩子，这么晚了，你去那儿干嘛？”

“见个人。”我说。

他又看了我一眼，没再问，挂挡起步。

车一路往北开，越走越荒，路灯也没了，只剩下黑漆漆的野地，还有风刮过空旷地面的声音。到地方，他踩了刹车。

我深吸了一口气，把手里攥得发热的那几张钱放在座位上：“不用找了，也不用等我。”

我伸手去推车门，没推开。又推了一下，还是不动。

“师傅，开下门。”我有点急。

他没动，啪嗒一声点了根红梅烟。火光一下子亮了，把他那张胡子拉碴的脸照得清清楚楚。他吸了一口烟，慢慢吐出来，说话也慢了下来：

“大姐，这底下哪有人啊？就一条河。大冷天的，你这是想下去游泳？还是给我添晦气？”

我一下子愣住了。眼泪在眼眶里打转，却怎么也掉不下来，嘴里只会一遍一遍地说：“你开门，我要下车。”

我说得越多，他就越笃定。他死活不让我下车，骂了一句，说自己今天真是倒霉，问我家在哪儿，说免费拉我回去。

那一刻我再也撑不住了，哭出了声。我跟他说，我没家了，什么都没了，我不该活着。我求他，放我下车。

他听完沉默了一会儿，忽然冷笑了一声：“想死？行，我不拦着你了。但账得先结清。”

“这趟来回十几公里，今晚算是彻底没法拉客了，再加上你这一身晦气弄得我心里不舒坦，精神损失费算两百块，不多吧？”

说完，他把车一掉头。

“我这车一睁眼就是公司的份子钱，钱没结清之前，你哪儿也别想去。阎王爷那儿，也不收欠账的。”

就这么着，我被他“绑”回了家。

那是一间小平房，也就十来平方米，门口堆着一堆黑乎乎的蜂窝煤，屋里是常年散不掉的机油味。灯泡昏黄，墙皮都起了卷儿。

他让我在墙角那张折叠床上凑合一宿，自己点了根烟，闷声问我：“以前干啥的？”

“裁缝。”我低着头，看着指甲缝里的黑泥，“开过店。后来……给孩子治病，都卖了。”

他没接话。

过了一会儿，他弯腰从床底拖出个鼓鼓囊囊的蛇皮袋，哗啦一声倒在地上。里面全是旧衣服，有的缺扣子，有的破了边，还有一条红丝绒围巾，被烟头烫了个洞。

“会做活儿就行。”他说，“这些修修补补，改改样式，兴许能卖几个钱。什么时候挣够那两百块还我，你爱去哪儿去哪儿。”

转眼天就冷了，雪下得特别大，整座城都白了。

他不知从哪儿又弄来些布料，还有一台掉了漆的“蝴蝶牌”缝纫机。踏板踩下去，哒哒响，声音不大，却很稳，像是给人定心用的。

白天他出去跑车，我就一个人在灯下干活。拆、洗、缝、拼，那条破了洞的丝绒围巾，被我改成了一个小手包；旧毛线帽绣上了野山菊；碎布头拼起来，做成发带——那阵子城里刚开始流行这些。

他的红色夏利，就成了我的流动橱窗。第一笔买卖是我钩的一顶缀着假珍珠的贝雷帽，卖了二十块。

晚上回来，他把那张还带着体温的票子往缝纫机上一拍，粗糙的手不小心碰到我的指尖，又立刻缩了回去，只闷闷地说了一句：“手艺还行，没给咱车丢人。”

那一瞬间，我看见他眼里闪了一下，比车里的暖风还热。

日子就这么过着，像缝纫机底下的线，一针一线，慢慢往前走。

除夕那天，外头鞭炮响个不停，硝烟味顺着门缝往里钻。他早早收了车，拎回来一袋速冻饺子，还有个用旧报纸包着的小东西。

“百货大楼门口练摊的，看着挺亮堂。”他说着，把东西递给我，手一直在搓，“就……顺手买的。”

是一枚红色的塑料发卡，亮得晃眼。

“头发老挡眼睛。”他说完就去拧收音机。

收音机里正放着春节联欢晚会的开场曲，热热闹闹的，把窗外的风雪声都盖住了。

我把发卡别在耳边，对着那面裂了缝的小圆镜看了又看。镜子里的女人，眼角的皱纹深了，可眼神不再是死的。

眼泪一滴一滴落在红丝绒上，晕开一小片深色。

我在心里轻声跟妮儿说：“孩子，再等等。妈现在，还不能去陪你。”
