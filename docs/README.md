# 在数字荒原里，捡拾微光

人们常说，互联网是有记忆的。但那份记忆往往只偏爱大人物、大事件，偏爱那些被流量与聚光灯反复照亮的瞬间。而我们这些草根，在虚拟世界里留下的悲欢离合，往往比在现实中留下的痕迹消散得更快。

多年以前，我曾在各大网络论坛里激扬文字。然而，当时代的潮水退去，那些曾经喧嚣的论坛一个接一个地关停、消失。服务器断电的那一刻，无数人的青春絮语、生活感悟，以及深夜敲下的诗句与牢骚，顷刻之间化为乌有。我自己写下的只言片语，也因未曾备份而再不可寻，仿佛我曾经来过的证据，都已被时间悄然抹去。

或许是命运的某种补偿，前些日子整理旧硬盘时，我意外发现了一些被岁月封存的文件。那是二十多年前，我无意中保存下来的一些帖子。作者都是匿名的普通网友，讲述的是发生在上个世纪的琐碎往事。

如果说历史是一条奔涌向前的大河，那么被写入史册的大人物就是翻涌在水面上的浪花；而普通人则是沉积在河床深处的沙砾，微小、无名，却承载着河流本身的重量。既然在机缘巧合下，我拾起了几颗被岁月淘洗出来的“沙砾”，便不忍看它们再次湮没于寂静。因此，我将这些故事重新编辑整理，发表于此。希望它们有机会穿透水面，被人看见。

编选此集，既是对那个一去不返的论坛时代的纪念，也是想借此证明：每一个平凡的生命，都曾在尘世中亮起过属于自己的微光，即便无人围观，即便转瞬即逝。

是为序。
